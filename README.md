## Basic Requirements

Your device should meet the following basic requirements.

```shell
ESP IDF: 5.5.0+
```

## Configuration

1、Select the type of target development board, for example: `esp32s3`

2、Find the `GEEKROS` configuration option in `menuconfig` and complete the relevant configurations.
